Hello welcome to squid game 1 - Red light, green light 

made in unity 2020.2.4f1.

How to download:
1. download apk file
2. upload apk file to oculus quest
3. play 

apk file is available on google drive : https://drive.google.com/file/d/1VfveTXKt5PGV3hvBXlEhTIndfYjWEQge/view?usp=sharing

How to play 
1. select y for menu, x to press "walk" key
2. use left controller to walk 

Win/Lose condition
1. Win: enter the red area behind the giant mask
2. Lose: Moving when the giant mask is facing you
